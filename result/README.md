# result/

Lưu kết quả kiểm thử mỗi lần chạy.

- Quy ước tên tệp: `result_<menu>_<MMDD>_<HHmm>.md`
  - Ví dụ: `result_home_0625_1733.md`
- Mỗi tệp gồm: bảng trạng thái từng mục checklist (PASS/FAIL/BLOCKED/N/A), phần tổng hợp số lượng, và danh sách lỗi kèm đường dẫn ảnh trong `screenshot/`.

Tệp `result_home_0625_1733.md` là **mẫu định dạng** tham khảo.
